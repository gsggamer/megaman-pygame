# megaman-pygame

## Pré-requisitos

Tenha certeza de ter os seguintes requisitos:

- Python 3 ou superior
- Pygame
- Joystick USB

## Como rodar o projeto:

Após clonar o repositório na sua máquina, rode o seguinte comando na raíz de seu projeto:

```python3 mega_data.py```
